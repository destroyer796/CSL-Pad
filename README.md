# CSL-Pad
A Hackpad made for Hack Club designed to be used with a Fanatec CDL DD wheelbase

## BOM:
 - 16x Cherry MX switches
 - 1x EC11 Encoder
 - 1x PCB
 - 17x Through-hole 1N4148 Diodes
 - 16x WS2812B LEDs
 - 1x XIAO RP2040
 - 1x 3D Printed Case
 - 16 DSA Keycaps
 - 1x 3D Printed Knob
 - 7x M3x5x4 Heatset Inserts
 - 3x M3x6 SHCS Screws
 - 4x M3x12 SHCS Screws
 - 2x M6x16 SHCS Screws
